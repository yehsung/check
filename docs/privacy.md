# 개인정보 원칙

`check`는 팀원이 직접 누른 근무 시작/종료와 팀 상태 표시만 다룹니다.

## 수집하는 데이터

- 이메일
- 표시 이름
- 팀 멤버십
- 근무 시작/종료 시각
- 현재 근무 상태
- 프로필 아바타 이미지(선택 — 팀원이 직접 고른 경우에만 공개 아바타 버킷에 저장)

로그인 비밀번호는 Supabase Auth 요청에만 사용하며 앱 저장소에 저장하지 않습니다. 다음 실행 때 편하게 이어 쓰도록 이메일, 별명, 그리고 로그인 세션 정보(액세스/리프레시 토큰과 사용자 식별자 UUID)는 이 Mac의 사용자 기본값(UserDefaults)에 저장됩니다. 세션이 만료되어 재로그인이 필요해지면 저장된 세션 정보는 지워집니다.

## 수집하지 않는 데이터

- 스크린샷
- 키보드 입력
- 마우스 움직임
- 앱/웹사이트 사용 기록
- 카메라, 마이크, 위치 정보
- 생산성 점수

앱은 Accessibility, Screen Recording, Camera, Microphone, Location 권한을 요청하지 않아야 합니다.
