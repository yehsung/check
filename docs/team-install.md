# check 설치 · 사용 안내 (팀원용)

`check`는 Mac 상단바(메뉴바)에서 근무 상태와 타이머를 확인하는 작은 앱입니다. 아래 순서대로만 따라 하면 됩니다.

## 1. 설치하기

1. 전달받은 `check.zip`을 더블클릭해 압축을 풉니다.
2. 나온 `check.app`을 **응용 프로그램(Applications)** 폴더로 끌어다 놓습니다.
3. `check.app`을 더블클릭해 실행을 시도합니다.
   - "확인되지 않은 개발자" 경고가 뜨며 열리지 않습니다. 정상입니다. **완료**(또는 취소)를 눌러 경고를 닫습니다.
4. **시스템 설정 > 개인정보 보호 및 보안**을 열고, 아래쪽에 나오는 check 안내에서 **그래도 열기**를 누릅니다.
   - 다시 확인 창이 뜨면 한 번 더 **열기**를 누릅니다. 이후로는 그냥 더블클릭으로 실행됩니다.
5. macOS 14(Sonoma) 이하라면 `check.app`을 **우클릭(또는 control + 클릭) → 열기 → 다시 열기**로도 첫 실행을 할 수 있습니다. macOS 15(Sequoia)부터는 이 우회가 막혀 4번 경로를 써야 합니다.

설치가 끝나면 화면 오른쪽 위 상단바에 check 아이콘이 나타납니다. 아이콘을 클릭하면 작은 창(팝오버)이 열립니다.

## 2. 처음이라면: 가입

상단바 아이콘을 클릭해 창을 연 뒤:

1. **별명**: 팀원 목록에 보일 이름(예: 홍길동)
2. **이메일**: 본인 이메일 주소
3. **비밀번호**: 앞으로 로그인에 쓸 비밀번호
4. **가입** 버튼을 누릅니다.

가입하면 자동으로 로그인되고, 별도의 팀 선택 없이 바로 팀에 들어갑니다.

## 3. 이미 계정이 있다면: 로그인

이메일과 비밀번호를 입력하고 **로그인** 버튼을 누릅니다. 별명은 처음 가입 때 정한 것이 그대로 쓰입니다.

## 4. 근무 시작 / 종료

- 창을 열면 오른쪽에 **근무 시작** 버튼이 있습니다. 누르면 타이머가 시작됩니다.
- 근무 중에는 버튼이 **근무 종료**로 바뀝니다. 일을 마치면 눌러서 종료합니다.
- 시작/종료는 팀 전체에 동기화되어, 다른 팀원의 창에도 내 상태가 보입니다.

## 5. 상단바 표시 읽는 법

상단바 아이콘과 옆의 글자로 지금 상태를 알 수 있습니다.

- **달리는 사람 아이콘 + 시간(예: 01:23)**: 근무 중이며, 지금까지 일한 시간입니다. (한 시간이 넘으면 `시:분`, 그 전에는 `분:초`)
- **일시정지 아이콘 + `오프`**: 근무 종료 상태입니다.
- **구름 느낌표 아이콘 + `대기`**: 서버와 동기화가 아직 안 끝난 상태입니다. 잠시 기다리면 됩니다.

창 안에서는 팀 이름과 함께 각 팀원이 근무 중인지, 현재/이번 주 근무시간이 얼마인지도 볼 수 있습니다. 창 하단에는 아이콘 버튼 3개가 나란히 있습니다: **새로고침**(원형 화살표), **로그아웃**(문에서 나가는 화살표), **앱 종료**(전원 아이콘). 각 아이콘에 마우스를 올리면 한글 툴팁이 떠서 어떤 버튼인지 알 수 있습니다.

## 6. 자주 묻는 문제

- **연결 오류가 나요 / 상태가 안 바뀌어요**
  서버(무료 플랜)가 한동안 미사용으로 잠시 멈춘 것일 수 있습니다. 담당자에게 "Supabase 프로젝트 Restore가 필요할 수 있다"고 알려 주세요. 복원되면 다시 정상 동작합니다.
- **`다시 로그인 필요`라고 떠요**
  로그인 세션이 만료된 것입니다. 이메일과 비밀번호를 다시 입력해 **로그인**하면 됩니다.
- **`Supabase 키 오류` / `Supabase 키 필요`가 떠요**
  앱 설정 값 문제입니다. 배포 담당자에게 문의하세요.
- **앱을 완전히 끄고 싶어요**
  창 하단의 **앱 종료** 아이콘 버튼(전원 아이콘, 마우스를 올리면 "앱 종료" 툴팁)을 누르면 check가 완전히 종료됩니다. 다시 쓰려면 응용 프로그램 폴더에서 `check`를 다시 실행하면 됩니다.
- **다른 계정으로 바꾸고 싶어요**
  창 하단의 **로그아웃** 아이콘 버튼(문에서 나가는 화살표, 마우스를 올리면 "로그아웃" 툴팁)을 누른 뒤, 다른 이메일/비밀번호로 로그인하면 됩니다.
